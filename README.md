# Observability Lab (POC)

This repository contains a complete observability stack using:

- OpenTelemetry Collector
- Prometheus
- Grafana
- Jaeger
- Splunk Enterprise & Splunk Observability Cloud

## Features
- Metrics, logs, and traces via OpenTelemetry
- Prometheus + Grafana dashboards
- Jaeger and Splunk APM for tracing
- Designed for Docker-based deployments

## Phases
- Phase 1: Metrics (Prometheus, Grafana)
- Phase 2: Exporters & Infra visibility
- Phase 3: Traces (Jaeger)
- Phase 4: Splunk Observability integration

## Usage
Each component can be started independently via Docker Compose.
